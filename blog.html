How to create your own web page?

Bagaimana langkah-langkah membuat halaman web dengan GIT?

1. Login GitHub
2. Membuat repositori baru
3. Buka aplikasi GIT Bash
4. Me-remote repository yang ada di GitHub
5. Meng-commit file HTML, CSS, dan lainnya
6. Push file yang sudah dicommit tadi ke dalam repositori GitHub
7. Cek status dengan GIT
8. Cek repositori GitHub
9. Atur bagian GitHubPages
10. Link web sudah dapat diakses
